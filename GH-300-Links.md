A consolidated learning resource repository for **GitHub Copilot training participants**.  
This README contains official learning paths, hands-on labs, documentation, certification resources, and advanced references to help you master GitHub Copilot and the broader GitHub ecosystem.

# GitHub Copilot

GitHub Copilot is an AI-powered developer tool that helps accelerate software development through:

- AI-assisted code completion
- Code generation
- Intelligent chat-based assistance
- Refactoring recommendations
- Test generation
- Agent-based automation
- Context-aware development workflows

🔗 **Official GitHub Copilot**  
https://github.com/features/copilot

# GitHub Learning Path

Progress through the GitHub learning ecosystem in the recommended order:

## 1. GitHub Foundations (GH-900)
Build foundational GitHub knowledge.

https://learn.microsoft.com/en-us/training/courses/gh-900t00

---

## 2. GitHub Administration Basics (GH-100)
Learn repository governance, administration, and collaboration management.

https://learn.microsoft.com/en-us/training/courses/gh-100t00

---

## 3. Automate Your Workflow with GitHub Actions (GH-200)
Master CI/CD automation with GitHub Actions.

https://learn.microsoft.com/en-us/training/courses/gh-200t00

---

## 4. GitHub Copilot (GH-300)
Develop AI-assisted software engineering skills using GitHub Copilot.

https://learn.microsoft.com/en-us/training/courses/gh-300t00

---

## 5. GitHub Advanced Security (GH-500)
Learn code scanning, secret scanning, and supply chain security.

https://learn.microsoft.com/en-us/training/courses/gh-500t00

Follow this sequence for best results:

### Phase 1 — Foundations
- GitHub Basics
- Repository Management
- Branching & Collaboration

### Phase 2 — Automation
- GitHub Actions
- CI/CD Workflows

### Phase 3 — Copilot Essentials
- Prompting
- Chat Modes
- Code Suggestions

### Phase 4 — Advanced Copilot
- Agent Mode
- Plan Mode
- MCP Integration
- Spaces

### Phase 5 — Certification Preparation
- Labs
- Practice Assessment
- Mock Tests
